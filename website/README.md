# chessgame 网页版

本目录是 `chessgame` 的本地网页部署版本。网页中的六种棋全部在浏览器内运行，不会启动或嵌入本地 EasyX 程序；现有 C++ 源码与本地可执行文件保持不变。

## 启动

双击：

`exe/chessgame_web.exe`

启动窗口会显示实际地址，默认是：

`http://127.0.0.1:8765`

若 8765 端口已被占用，程序会自动尝试后续端口。保持启动窗口开启，在浏览器中访问其显示的网址即可。关闭启动窗口后，本地网站停止运行。

也可以直接使用 Python 启动：

```powershell
python daima/backend/qilei_server.py
```

如不希望自动打开浏览器：

```powershell
python daima/backend/qilei_server.py --no-browser
```

## 技术与目录

- 前端逻辑：JavaScript
- 后端逻辑：Python
- 页面结构和样式：`daima/frontend/index.html`、`daima/frontend/css/main.css`
- 六种棋前端实现：`daima/frontend/js/games`
- 登录、存档和本地服务：`daima/backend`
- 用户运行数据：`data`
- 启动程序：`exe/chessgame_web.exe`

代码只保存在 `daima`，`data` 中没有程序源代码。

每个玩家的数据目录固定为：

```text
data/
└─ 用户名/
   ├─ user/
   │  └─ 用户名.json
   └─ result/
      ├─ 一局游戏.json
      └─ 另一局游戏.json
```

`user` 保存用户名和密码摘要，磁盘不保存明文密码；`result` 每一局一份 JSON，同时包含历史信息和该局存档。历史页面删除一条记录时，会同步删除对应 JSON 文件。

## 登录与存档

- 第一次使用先注册，用户名和密码都只允许数字。
- 用户名不能重复。
- 管理员账号按项目约定内置，但登录页面不会公开显示账号或密码。
- 所有用户都能在“个人中心”验证当前密码并修改自己的密码。
- 管理员能在“用户管理”中新增用户、修改用户密码、删除普通用户；删除时会同步移除该用户的全部数据文件。
- 每次打开一种棋，先选择“开始新游戏”或“读取上一次存档”。
- 没有可用存档时，“读取上一次存档”按钮为禁用状态。
- 退出游戏或关闭页面前，当前棋局会保存到该用户自己的 `result` 文件。

## 六种棋的操作

- 中国象棋、国际象棋：先选棋子，再选落点，最后确认；机器人也逐阶段显示相同过程。
- 围棋、五子棋：第一次选择只显示落点轮廓，再次确认才正式落子；机器人同样先预览再确认。
- 飞行棋：进入后保持左侧棋盘、右侧操作区；设置完成后先用滚动骰面抽签，确认顺序后再开始正式计时。设置页使用方向键调整；抽签和对局中按空格投骰；按 `1`～`6` 选择棋子，再按一次相同数字确认；按 `0` 跳过。棋子进入本队起点时不飞跃，之后落到本队普通色格才飞跃到下一同色格，并保留队伍颜色虚线路径；同一格上的多枚棋子会错位显示。步时或局时耗尽后固定显示“超时出局”及名次、完成数，整局结束时列出完整排名。
- 跳棋：进入后保持左侧星形棋盘、右侧操作区；设置页使用方向键调整；对局中按棋子编号选棋，随后按棋盘显示的落点编号预选，再按一次相同数字确认；目标选择阶段按 `0` 返回选棋。无合法走法、步时或局时耗尽均会判负出局；多人对局会继续到全部名次确定，并在终局列出完整排名。
- 飞行棋、跳棋的机器人沿用对应 C++ 版本的规则评价与决策思路；中国象棋、国际象棋、围棋和五子棋优先调用各自的外部棋力程序，程序不可用时自动使用内置算法继续对局。网页界面统一只显示“机器人”，不显示程序名称。
- 对局底部按键统一为“悔棋 (U)”“重开 (N)”“返回主界面 (Esc)”；飞行棋右侧保留独立骰子区域。
- 飞行棋和跳棋也可按 `Delete` 或 `Backspace` 悔棋；步时或局时耗尽会立即执行判负规则。
- 游戏内可使用右侧按钮返回主界面，支持的页面也可按 `Esc` 返回。
- 开局预测统一为 50/50（多人棋为等概率），产生有效行动后再根据局面更新。
- 五子棋、围棋的预测条左侧深色对应黑方、右侧浅色对应白方；中国象棋、国际象棋按各自先后方颜色显示。

首次进入围棋机器人对局时需要加载棋力文件，等待时间可能略长；加载完成后的后续决策会直接复用同一进程。

## 重新打包启动程序

安装 Python 与 PyInstaller 后，在 `website` 的上级目录执行：

```powershell
python -m PyInstaller --noconfirm --clean --onefile --console --name chessgame_web --paths website/daima/backend --distpath website/exe --workpath .build/pyinstaller --specpath .build website/daima/backend/qilei_server.py
```

打包完成后删除临时 `.build` 文件夹即可。启动程序依赖同级 `website/daima` 和 `website/data` 目录，移动时请整体移动 `website`。
